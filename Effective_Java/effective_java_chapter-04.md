# 클래스와 인터페이스

추상화의 기본 단위인 클래스와 인터페이스는 자바 언어의 심장과도 같음.

그래서 자바 언어에는 클래스와 인터페이스 설계에 사용하는 강력한 요소가 많이 있음.

이번 장에서는 이런 요소를 적절히 활용하여 클래스와 인터페이스를 쓰기 편하고, 견고하며, 유연하게 만드는 방법을 기술함.
